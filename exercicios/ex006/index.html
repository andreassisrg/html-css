Principais Formatações

Tags semânticas - Recomendadas

Tag STRONG para criar frases em negrito

Tag EM (emphasis) para criar frases em itálico

Tags não semânticas - Tendência ao desuso

Tag B (bold) para negrito

Tag I (italic) para itálico

Marcações no Texto

É possível marcar o texto por meio da tag MARK.

O estilo pode ser editado por dentro da tag ou por uma tag de estilo na "head"

Texto Deletado

Para tal, se usa a tag "del"

Texto inserido

Recomenda-se o uso da tag "ins" para marcar o texto como inserido, ou seja, sublinho

Obs: a tag "u" provavelmente entrará em desuso, por não ser semântica

Letras Pequenas e Grandes

Tags "small" para letras Pequenas como em contratos e "big" tag em desuso

Deslocamento inline

É possível usar a tag "sup" para criar coisas como x 20 + 3

E a tag "sub" para H20
